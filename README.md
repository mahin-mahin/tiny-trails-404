# TinyTrails 404 Error Page

A full-screen animated 404 error page for the TinyTrails children's brand.

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop
- **Animated Background**: Large "404" text with dynamic scaling and mask effects
- **Mobile Menu**: Slide-in menu with staggered animations
- **Video Integration**: Center video with mix-blend-darken effect
- **Smooth Transitions**: CSS transitions and animations throughout

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS 3
- Lucide React (icons)
- Vite (build tool)

## Getting Started

```bash
npm install
npm run dev
```
